T-T Nails & Spa Website

Files included:
- index.html
- styles.css
- script.js
- assets/ SVG logo, hero, gallery, services, favicon, OG image
- robots.txt
- sitemap.xml

How to publish:
Upload all files/folders to your hosting public_html or website root. Keep the assets folder beside index.html.

Before going live:
1. Replace https://ttnailsspa.com/ in index.html, robots.txt, and sitemap.xml if your domain is different.
2. Confirm phone number and hours are correct.
3. Add real nail photos later by replacing the SVG files in assets with JPG/PNG using the same filenames.
